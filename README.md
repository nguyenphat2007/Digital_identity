# Title
WorkID: Decentralized Competence Identity System

# Description
Traditional CVs and resumes are easily falsified or exaggerated. In the digital age, we need a trustless way to verify professional skills. I built this project to shift the focus from "who you are" to "what you can actually do." WorkID is a decentralized identity system built on the Stellar network (Soroban) where a user's skills and competencies are verified through a transparent, peer-endorsed mechanism rather than central authorities.

# Features
* **Profile Creation:** Users can initialize their digital identity on the blockchain with a custom display name and biography.
* **Peer-Endorsement System:** Users can endorse (vote for) the skills of others they have worked with.
* **Anti-Self-Endorsement:** The smart contract includes logic to prevent users from artificially inflating their own reputation score.
* **Dynamic Reputation Score:** A user's trust level grows strictly based on on-chain endorsements, making it a reliable metric for competencies.

# Future scopes
* **Web3 Recruitment Integration:** Creating an API to allow employers and job platforms to query a candidate's reputation score directly from the Stellar blockchain.
* **NFT Skill Badges:** Automatically issuing non-transferable Soulbound Tokens (SBTs) when users reach specific reputation milestones (e.g., 50 endorsements).
* **Decentralized CRM & Payroll:** Integrating the identity system into business management tools to automate payroll based on verified skill levels.
